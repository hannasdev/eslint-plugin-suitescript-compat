# Code Of Conduct

This project is a small open source repository for source-backed SuiteScript
compatibility tooling. Participation should stay practical, respectful, and
safe to share publicly.

## Expectations

- Be clear, constructive, and patient.
- Assume contributors may be working from different NetSuite account contexts
  or release constraints.
- Keep examples sanitized and minimal.
- Do not post credentials, tokens, account IDs, internal URLs, customer
  screenshots, proprietary scripts, or private implementation details.
- Do not use harassment, personal attacks, or discriminatory language.

## Enforcement

`@hannasdev` is the project maintainer for conduct decisions. The maintainer may
edit, hide, lock, or remove comments and issues; ask participants to revise
their behavior; or block participation when needed to keep the project usable.

Use public issues and pull requests for normal project discussion. Do not use
GitHub private vulnerability reporting for support requests, rule requests, or
ordinary conduct discussions.
